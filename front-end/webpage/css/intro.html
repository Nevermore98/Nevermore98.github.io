                  概述

                  2022年5月18日小于 1 分钟

                  层叠样式表 (Cascading Style Sheets，缩写为 CSS），是一种样式表语言（不是编程、标记语言），用来描述 HTML 或 XML 文档的样貌。目前广泛使用的版本 CSS2.1，CSS3正在标准化。

                  层叠指的是：

                  • 样式层叠：可以多次对同一选择器进行样式声明
                  • 选择器层叠：可以用不同选择器对同一个元素进行样式声明
                  • 文件层叠：可以用多个文件进行层叠

                  这些特性使得CSS极度灵活，导致其不具有正交性，修改一处可能导致多处修改
