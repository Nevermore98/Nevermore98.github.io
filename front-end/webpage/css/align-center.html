                                          注：把宽、高、边框、背景色作为内联样式，便于直观理解实现垂直和水平居中的 CSS 代码。

</div>

                                          # 3. 绝对定位

                                          需要提前知道 child 的尺寸

                                          （1）left: 50% + margin-left: -宽度的一半

                                          .parent {
  position: relative;
}
.child {
    position: absolute;
    left: 50%;
    margin-left: -50px; 
}

                                          （2）left/right: 0 + margin: auto

                                          .parent {
  position: relative;
}
.child {
    position: absolute;
    left: 0;
    right: 0;
    margin: auto;
}

                                          （3）left: 50% + transform

                                          .parent {
  position: relative;
}
.child {
  position: absolute;
  left: 50%;
  transform: translate(-50%, 0);
}

                                          # 3. 绝对定位

                                          需要提前知道 child 的尺寸

                                          （1）top: 50% + margin-top: -高度的一半

                                          .parent {
    position: relative;
}
.child {
    position: absolute;
    top: 50%;
    margin-top: -50px;
}

                                          （2）top/bottom: 0 + margin: auto

                                          .parent {
  position: relative;
}
.child {
  position: absolute;
  top: 0;
  bottom: 0;
  margin: auto;
}

                                          （3）top: 50% + transform

                                          .parent {
    position: relative;
}
.child {
    position: absolute;
    top: 50%;
    transform: translate(0, -50%);
}

                                          # 4. 伪元素

                                          :before、:after 必须都写

                                          .parent {
    text-align: center;
}
.child {
    display: inline-block;
    vertical-align: middle;
}
.parent::before {
    content: '';
    height: 100%;
    display: inline-block;
    vertical-align: middle;            
}
.parent::after {
    content: '';
    height: 100%;
    display: inline-block;
    vertical-align: middle;            
}
